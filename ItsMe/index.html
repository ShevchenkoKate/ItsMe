Создание макета сайта

Создание макета - это первое с чего начинается разработка сайта. С помощью него можно видеть и проработать визуальную составляющую сайта, зарание перед его созданием.

Вёрстка

Верстка сайта эта следующий этап создания сайта после создания макета. Она заключается в создании кода веб-страницы с помощью языка разметки (HTML) и оформлении того же кода с помощью каскадных таблиц стилей (CSS).
